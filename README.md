# Eventra

**Eventra** is a modern event ticketing platform built for creating, managing, and selling tickets online. It provides a clean, fast, and reliable experience for both event organizers and attendees, handling everything from event listings to secure payments.

Eventra is designed to be simple on the surface while supporting real-world marketplace functionality behind the scenes.

https://eventratickets.com
---

## What Eventra Does

* Allows organizers to create events and sell tickets online
* Lets users browse events and purchase tickets securely
* Handles payments, platform fees, and payouts automatically
* Issues tickets instantly after successful checkout
* Keeps ticket inventory accurate in real time

---

## Platform Overview

Eventra functions as a **two-sided marketplace**:

* **Organizers** list events and manage ticket availability
* **Attendees** discover events and purchase tickets
* **Eventra** facilitates payments and ensures a smooth transaction flow

The platform manages the full lifecycle of a ticket, from creation to payment to delivery.

---

## Technology

Eventra is built using modern web technologies, focusing on performance, type safety, and developer reliability:

* Next.js and React
* TypeScript
* Tailwind CSS
* Stripe for payments
* Clerk for authentication
* Convex for backend data and real-time updates
